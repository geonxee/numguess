# numguess

## Installation

```shell
$ pip install
```

## How to start

```shell
$ python numguess.py
```

## Features

- Get username
- Send a Greeting message with username
- Create random number
- Get user's guess
- Print user's guess
- if user's guess is random number, True
     user's guess is not random number, False
